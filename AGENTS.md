# Agents Handbook Instructions

This repository is the source of truth for reusable agentic workflows.

## Repository Layout

- `global/` contains global agent instructions.
- `skills/` contains reusable skill packages.
- `templates/` contains project-level instruction templates.
- `knowledge/` contains principles, patterns, and playbooks.
- `evaluations/` contains tests and rubrics for validating workflows.
- `scripts/` contains install and maintenance helpers.
